# Medical Image Processing (17.BMI121)

<!-- Serbian - English -->
[SRB](https://github.com/nebojsa-bozanic/BMI_OSuM/blob/master/README.md) &nbsp; [EN](https://github.com/nebojsa-bozanic/BMI_OSuM/blob/master/docs/en/README-EN.md)

This repository contains materials from practica for the course "Medical Image Processing", which is lectured by [prof. Dr. Vladimir Petrović](https://www.ktios.ftn.uns.ac.rs/profesori/vladimir-petrovic/) with assistants. The subject is part of bachelor's studies, course curriculum [Biomedical Engineering](http://www.ftn.uns.ac.rs/n1467299313/biomedicinsko-inzenjerstvo), IMT Studies (Biomedical Engineering: Electrical and Computer Engineering; Medical Sciences), Faculty of Technical Sciences in Novi Sad. The course is taken by 4th year students during the summer semester. The exercises are carried out in NTP, Lab115L.

The exercises are done in the programming interpreter language [Python](https://www.python.org/). Instead of classic editors, they are performed in [Google Colab](https://colab.research.google.com/), which can be run from any internet browser, while python notebooks can also be run locally.
